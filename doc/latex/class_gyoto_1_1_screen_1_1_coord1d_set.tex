\doxysubsection{Detailed Description}
Set of 1-\/d coordinates\+: indices or angles. 

Acts like a container (array-\/like) of either size\+\_\+t (pixel coordinate) or double (angle) values. This container can be iterated-\/through using the \mbox{\hyperlink{class_gyoto_1_1_screen_1_1_coord1d_set_a4673770278cc88ccfd96b4330c87af73}{operator++()}}, derefenced using the \mbox{\hyperlink{class_gyoto_1_1_screen_1_1_coord1d_set_afa9b992fd12a2229ed6144184a6806df}{operator$\ast$()}} (if containing pixel coordinates) or \mbox{\hyperlink{class_gyoto_1_1_screen_1_1_coord1d_set_a9b3a6aef00f09e8f918616f75ba86089}{angle()}} (in containing angles).
