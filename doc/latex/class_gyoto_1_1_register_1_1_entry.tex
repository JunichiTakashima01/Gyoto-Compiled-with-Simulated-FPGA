Get subcontractor for a given name. 

Search through the register for an \mbox{\hyperlink{class_gyoto_1_1_register_1_1_entry}{Entry}} matching name and return the corresponding subcontractor. If plugin is specified, only a subcontractor matching both name and plugin will be returned. Note that \mbox{\hyperlink{namespace_gyoto_1_1_astrobj_aff0e3a9c31318709959bb2471a22ff74}{Gyoto\+::\+Entry\+::get\+Subcontractor()}} will not load the plug-\/in for you, contrary to e.\+g. \mbox{\hyperlink{namespace_gyoto_1_1_metric_ac6c88d7fec42b5aa192e90b3a721993b}{Gyoto\+::\+Metric\+::get\+Subcontractor()}}. If plugin is the empty string, then the first subcontractor matching name will be returned, and the name of the plug-\/in it belongs to will be returned in plugin upon output.
