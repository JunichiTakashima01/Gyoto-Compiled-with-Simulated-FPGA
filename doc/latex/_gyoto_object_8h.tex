Declare virtual bool is\+Thread\+Safe() const. 

Use this to declare that the object is not (or not always) thread safe. The corresponding definition of is\+Thread\+Safe() must exist. If the object is always thread unsafe (e.\+g. Lorene Metrics of Python based objects), you can simply use GYOTO\+\_\+\+PROPERTY\+\_\+\+THREAD\+\_\+\+SAFETY in the corresponding .C file.
