\doxysubsection{Detailed Description}
Base class for metric description. 

Classes which represent a metric (e.\+g. Gyoto\+::\+Kerr) should inherit from \mbox{\hyperlink{class_gyoto_1_1_metric_1_1_generic}{Gyoto\+::\+Metric\+::\+Generic}} and implement all of the virtual methods plus at least one of the gmunu methods and one of the christoffel methods.
