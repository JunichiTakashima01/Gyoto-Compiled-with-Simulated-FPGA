\doxysubsection{Detailed Description}
Reference-\/counting pointers. 

Template class Gyoto\+::\+Smart\+Pointer$<$\+T$>$ performs reference counting on its pointee and makes sure the pointed to object is actually deleted when the number of references to it drops to zero. The pointed-\/to object must inherit from \mbox{\hyperlink{class_gyoto_1_1_smart_pointee}{Gyoto\+::\+Smart\+Pointee}} for this to work and be referenced to only by Smart\+Pointers.
