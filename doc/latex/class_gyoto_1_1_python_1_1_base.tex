\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_gyoto_1_1_python_1_1_base}{Base}} class for classes in the \mbox{\hyperlink{namespace_gyoto_1_1_python}{Python}} plug-\/in. 

All derived C++ classes (\mbox{\hyperlink{class_gyoto_1_1_metric_1_1_python}{Gyoto\+::\+Metric\+::\+Python}}, \mbox{\hyperlink{class_gyoto_1_1_spectrum_1_1_python}{Gyoto\+::\+Spectrum\+::\+Python}}, \mbox{\hyperlink{class_gyoto_1_1_astrobj_1_1_python_1_1_standard}{Gyoto\+::\+Astrobj\+::\+Python\+::\+Standard}} and Gyoto\+::\+Astrobj\+::\+Python\+::\+Thindisk have those three Properties\+:
\begin{DoxyItemize}
\item Module (string)\+: the module in which the \mbox{\hyperlink{namespace_gyoto_1_1_python}{Python}} class is implemented;
\item Class (string)\+: the name of the \mbox{\hyperlink{namespace_gyoto_1_1_python}{Python}} class, in module Module, to interface with;
\item Parameters (vector$<$double$>$)\+: optional list of parameters for this class. These parameters are passed one by one to the \mbox{\hyperlink{namespace_gyoto_1_1_python}{Python}} instance using {\bfseries{setitem}} with numerical keys.
\end{DoxyItemize}

Those C++ classes wrap around \mbox{\hyperlink{namespace_gyoto_1_1_python}{Python}} classes that implement certain methods. Those \mbox{\hyperlink{namespace_gyoto_1_1_python}{Python}} classes may accepts parameters through the Parameters property described above by implementing {\bfseries{setitem}}, or by providing\+:
\begin{DoxyItemize}
\item member \char`\"{}properties\char`\"{}, a dict in the form \{name\+: dtype\} where name is the property name and dtype the corresponding datatype.
\item methods set(self, key, value) and get(self, key).
\end{DoxyItemize}

All the \mbox{\hyperlink{namespace_gyoto}{Gyoto}} instances of the classes descending from \mbox{\hyperlink{class_gyoto_1_1_python_1_1_base}{Gyoto\+::\+Python\+::\+Base}} expose themselves to the \mbox{\hyperlink{namespace_gyoto_1_1_python}{Python}} instance they wrap immediately after instantiation by setting the \textquotesingle{}this\textquotesingle{} attribute. If the \textquotesingle{}gyoto\textquotesingle{} \mbox{\hyperlink{namespace_gyoto_1_1_python}{Python}} extension can be loaded, then \textquotesingle{}this\textquotesingle{} will be an instance of one of the classes gyoto.\+Metric, gyoto.\+Spectrum, gyoto.\+Standard\+Astrobj or gyoto.\+Thin\+Disk pointing to the underlying C++ instance. If the \textquotesingle{}gyoto\textquotesingle{} extension is not available, \textquotesingle{}this\textquotesingle{} will be None.
