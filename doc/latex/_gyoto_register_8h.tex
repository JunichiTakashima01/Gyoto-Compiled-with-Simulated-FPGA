\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_gyoto}{Gyoto}} registers. 

\mbox{\hyperlink{class_gyoto_1_1_register_1_1_entry}{Gyoto\+::\+Register\+::\+Entry}} instances are used to map kind names to \mbox{\hyperlink{class_gyoto_1_1_smart_pointee_afe17bea3c33e2187940cfdd5543c1d14}{Gyoto\+::\+Smart\+Pointee\+::\+Subcontractor\+\_\+t}} functions used to instantiate objects from XML files through the \mbox{\hyperlink{class_gyoto_1_1_factory}{Gyoto\+::\+Factory}}.
