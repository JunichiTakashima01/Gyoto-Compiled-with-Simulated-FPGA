\doxysubsection{Detailed Description}
Tellers tell Listeners when they mutate. 

A Listener can hook() to a Teller. The Teller will tell it when it mutates using Listener\+::tell(), usually through the highter lever Teller\+::tell\+Listeners(). The Listener can later unhook(). The Listener must therefore implement Listener\+::tell() to react when it is told.
