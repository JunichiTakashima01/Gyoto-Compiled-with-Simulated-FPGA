\doxysubsection{Detailed Description}
Access to spectrometers. 

Objects which describe spectrometers (including one-\/channel devices, a.\+k.\+a cameras) must inherit from the \mbox{\hyperlink{class_gyoto_1_1_spectrometer_1_1_generic}{Gyoto\+::\+Spectrometer\+::\+Generic}} class.

To be usable, a \mbox{\hyperlink{class_gyoto_1_1_spectrometer_1_1_generic}{Spectrometer\+::\+Generic}} sub-\/class should register a \mbox{\hyperlink{namespace_gyoto_1_1_spectrometer_a63d8c94e41e4be8a817c095b457de8a5}{Spectrometer\+::\+Subcontractor\+\_\+t}} function using the \mbox{\hyperlink{namespace_gyoto_1_1_spectrometer_a86b5778d46f54fc06ff22bc060987c6b}{Spectrometer\+::\+Register()}} function. See also \mbox{\hyperlink{writing_plugins_page}{Writing plug-\/ins for Gyoto}} .

Type for \mbox{\hyperlink{namespace_gyoto_1_1_spectrometer}{Spectrometer}} kind. 

\mbox{\hyperlink{namespace_gyoto_1_1_spectrometer}{Spectrometer}} kindid is a unique numerical identifier for that kind, produced as the address to a static C string variable holding the kind\textquotesingle{}s name. Most of the time, the address is the only significant part as this is more reliable and allows for direct numerical comparison instead of slower string comparison. The value of the string variable can be used for printing and as anchor name for \mbox{\hyperlink{namespace_gyoto_1_1_spectrometer_a86b5778d46f54fc06ff22bc060987c6b}{Register()}}, although the anchor name could be different. \mbox{\Hypertarget{namespace_gyoto_1_1_spectrometer_a63d8c94e41e4be8a817c095b457de8a5}\label{namespace_gyoto_1_1_spectrometer_a63d8c94e41e4be8a817c095b457de8a5}}
