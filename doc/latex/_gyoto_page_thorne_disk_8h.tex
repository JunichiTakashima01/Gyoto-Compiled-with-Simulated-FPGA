\doxysubsection{Detailed Description}
A geometrically thin, optically thick disk. 

This class describes a disk contained in the z=0 (equatorial) plane, extending from r=r\+\_\+\+ISCO to r=infinity. The flux emitted at radius r is given by Page \& Thorne (1974, ApJ 191\+:499, Eqs. 11b, 14, 15).

Only bolometric flux is implemented (as quantity User4), no spectral resolution.
