\doxysubsection{Detailed Description}
Timelike or null geodesics. 

Their are two derived classes\+: \mbox{\hyperlink{class_gyoto_1_1_photon}{Photon}} and Star. A \mbox{\hyperlink{class_gyoto_1_1_worldline}{Worldline}} can be integrated from an initial condition either backward or forward in time using \mbox{\hyperlink{class_gyoto_1_1_worldline_ab0b41f07994311e3c107a4ec06c18b76}{x\+Fill()}} (\mbox{\hyperlink{class_gyoto_1_1_photon_a5f6f70601d8581ffbbe11b677b20417f}{Photon\+::hit()}} also integrates the \mbox{\hyperlink{class_gyoto_1_1_worldline}{Worldline}}). Member \mbox{\hyperlink{class_gyoto_1_1_worldline_a3edacdf2600826da936645fe00ad014e}{state\+\_\+}} holds the integration state as well as an integrator. There are several kinds of integration states, that derive from \mbox{\hyperlink{class_gyoto_1_1_worldline_1_1_integ_state_1_1_generic}{Integ\+State\+::\+Generic}}.

The coordinates of the \mbox{\hyperlink{class_gyoto_1_1_worldline}{Worldline}} are stored in \mbox{\hyperlink{class_gyoto_1_1_worldline_ac57e450d22b0545edeca560416fd052e}{x0\+\_\+}}, \mbox{\hyperlink{class_gyoto_1_1_worldline_a085a68b8f8cdaf7bd03a8aa2b5f3a6e2}{x1\+\_\+}}, \mbox{\hyperlink{class_gyoto_1_1_worldline_a6ffee8d466efc462d086bbdced97b4e2}{x2\+\_\+}}, \mbox{\hyperlink{class_gyoto_1_1_worldline_aa48ffa28ea27f2ad22f158cc00ea959c}{x3\+\_\+}}, \mbox{\hyperlink{class_gyoto_1_1_worldline_a0fac003cf86d5c632b3706aaace5296a}{x0dot\+\_\+}}, \mbox{\hyperlink{class_gyoto_1_1_worldline_a0f66fed8a931456a6d629543a08c04de}{x1dot\+\_\+}}, \mbox{\hyperlink{class_gyoto_1_1_worldline_afc6d6fedffc0bc06c9ce95eb34acbe7e}{x2dot\+\_\+}} ans \mbox{\hyperlink{class_gyoto_1_1_worldline_a18fd3d49a9bd515e5bf9685f3db98e16}{x3dot\+\_\+}}. Those arrays are extended as needed using \mbox{\hyperlink{class_gyoto_1_1_worldline_ac5643cc01a45e5041fe7e331c663226f}{x\+Expand()}}. These coordinates can be retrieved using \mbox{\hyperlink{class_gyoto_1_1_worldline_a9bf6ee964374561bb01ea3ab218d9c9c}{get\+\_\+t()}}, \mbox{\hyperlink{class_gyoto_1_1_worldline_a6b1c7ba28dfdf02f6bbafb8af5bc1025}{get\+\_\+xyz()}}, \mbox{\hyperlink{class_gyoto_1_1_worldline_a52e534430d89982b4b617e51e7aa0807}{get\+Cartesian()}}, \mbox{\hyperlink{class_gyoto_1_1_worldline_aa719a09c91b55c2fb239ec9dc2d59c21}{get\+Coord()}} etc.

\mbox{\hyperlink{class_gyoto_1_1_worldline}{Worldline}} does not derive from \mbox{\hyperlink{class_gyoto_1_1_object}{Object}}, and does not instantiate a \mbox{\hyperlink{class_gyoto_1_1_property}{Property}} list. This is because this would lead to multiple inheritance of the \mbox{\hyperlink{class_gyoto_1_1_object}{Object}} base in derived classes. Instead, \#\+Gyoto\+Worldline.\+h provides a few macros that can be used to include the \mbox{\hyperlink{class_gyoto_1_1_worldline}{Worldline}} properties in a derived classe\textquotesingle{}s \mbox{\hyperlink{class_gyoto_1_1_property}{Property}} list\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{_gyoto_worldline_8h_ad22d501635a94e86f28ee2adab6242c8}{GYOTO\+\_\+\+WORLDLINE}} is to be used in a public section of the derived class declaration (.h file); it declares wrappers around the \mbox{\hyperlink{class_gyoto_1_1_worldline}{Worldline}} property accessors;
\item \mbox{\hyperlink{_gyoto_worldline_8h_a92ffb7845551c3f7812bfbd5e69f3c9a}{GYOTO\+\_\+\+WORLDLINE\+\_\+\+ACCESSORS}} is to be used with the class definition (.C file; it defines the accessors declared by \mbox{\hyperlink{_gyoto_worldline_8h_ad22d501635a94e86f28ee2adab6242c8}{GYOTO\+\_\+\+WORLDLINE}};
\item \mbox{\hyperlink{_gyoto_worldline_8h_aa765beed747e6fb92152b6188097dba7}{GYOTO\+\_\+\+WORLDLINE\+\_\+\+PROPERTIES}} declares the Properties that use these accessors. It must be used like e.\+g. \mbox{\hyperlink{_gyoto_property_8h_a51b9b6d83536618f8c5c04101168305e}{GYOTO\+\_\+\+PROPERTY\+\_\+\+DOUBLE}}, between \mbox{\hyperlink{_gyoto_property_8h_add8e16ed1e17ab2683b08cc330b341d4}{GYOTO\+\_\+\+PROPERTY\+\_\+\+START}} andf \mbox{\hyperlink{_gyoto_property_8h_a1356435ea7ee0e18d07201a80e034dcc}{GYOTO\+\_\+\+PROPERTY\+\_\+\+END}}.
\item Finally, \mbox{\hyperlink{_gyoto_worldline_8h_a1be18e74aa4c1176ff9bcb6e72cc3faf}{GYOTO\+\_\+\+WORLDLINE\+\_\+\+PROPERTY\+\_\+\+END}} is a drop-\/in replacement for \mbox{\hyperlink{_gyoto_property_8h_a1356435ea7ee0e18d07201a80e034dcc}{GYOTO\+\_\+\+PROPERTY\+\_\+\+END}} that calls \mbox{\hyperlink{_gyoto_worldline_8h_aa765beed747e6fb92152b6188097dba7}{GYOTO\+\_\+\+WORLDLINE\+\_\+\+PROPERTIES}} and \mbox{\hyperlink{_gyoto_worldline_8h_a92ffb7845551c3f7812bfbd5e69f3c9a}{GYOTO\+\_\+\+WORLDLINE\+\_\+\+ACCESSORS}}.
