\doxysubsection{Detailed Description}
All of \mbox{\hyperlink{namespace_gyoto}{Gyoto}}. 

\mbox{\hyperlink{_gyoto_8h}{Gyoto.\+h}} should include all of the other \mbox{\hyperlink{namespace_gyoto}{Gyoto}} C++ header files. Note that at any given time it may be lacking some new headers. It is always best to specifically include the headers that you actually use.

Beware that some symbols are resolved only in the plugins (libgyoto-\/stdplug.\+so, libgyoto-\/lorene.\+so).
