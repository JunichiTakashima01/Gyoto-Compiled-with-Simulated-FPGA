\doxysubsection{Detailed Description}
Base class for work in progress. 

The constructors of this class simply issue a warning that the (derived) class is work in progress.

Issue a warning specifying the name of the derived class. 

If classname is the empty string (\char`\"{}\char`\"{}), the warning is not issued. Use this to mark that a class is no more work in progress without breaking the ABI (i.\+e. in the \mbox{\hyperlink{namespace_gyoto}{Gyoto}} stable branch).
