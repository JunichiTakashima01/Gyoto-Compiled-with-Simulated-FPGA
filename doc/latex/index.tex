\mbox{\hyperlink{namespace_gyoto}{Gyoto}} aims at providing a framework for computing orbits and ray-\/traced images in General relativity. It consists of a library (libgyoto), utility programs, a plug-\/in for the Yorick programing language, and an extension for the Python 3 programing language. \mbox{\hyperlink{namespace_gyoto}{Gyoto}} is known to run under Linux and Mac OS X. Please do tell us if you manage to run \mbox{\hyperlink{namespace_gyoto}{Gyoto}} under a different OS. It should compile and run with moderate effort on most UNIX-\/like systems.

\mbox{\hyperlink{namespace_gyoto}{Gyoto}} can be expanded with plug-\/ins providing custom \mbox{\hyperlink{class_gyoto_1_1_metric_1_1_generic}{Gyoto\+::\+Metric\+::\+Generic}}, \mbox{\hyperlink{class_gyoto_1_1_astrobj_1_1_generic}{Gyoto\+::\+Astrobj\+::\+Generic}} and \mbox{\hyperlink{class_gyoto_1_1_spectrum_1_1_generic}{Gyoto\+::\+Spectrum\+::\+Generic}} classes, which describe respectively analytical or numerical metrics, astronomical objects, and spectral shapes for astronomical objects (see \mbox{\hyperlink{writing_plugins_page}{Writing plug-\/ins for Gyoto}}). Custom \mbox{\hyperlink{class_gyoto_1_1_metric_1_1_generic}{Gyoto\+::\+Metric\+::\+Generic}}, \mbox{\hyperlink{class_gyoto_1_1_astrobj_1_1_generic}{Gyoto\+::\+Astrobj\+::\+Generic}} and \mbox{\hyperlink{class_gyoto_1_1_spectrum_1_1_generic}{Gyoto\+::\+Spectrum\+::\+Generic}} classes can also be written in the \href{https://www.python.org/}{\texttt{ Python}} 3.\+x interpreted language using the {\ttfamily python} plug-\/in for \mbox{\hyperlink{namespace_gyoto}{Gyoto}}, see below. Beware that a Python implementation of a custom class will run significantly slower than the equivalent C++ implementation, but sometimes saving on development time is better than saving on computing time.

The base distribution includes three plug-\/ins\+: the standard plug-\/in ({\ttfamily stdplug}), a plug-\/in for using \href{http://www.lorene.obspm.fr/}{\texttt{ LORENE}}-\/based numerical metrics ({\ttfamily lorene}) and a plug-\/in for writing custom astronomical objects or metrics in the \href{https://www.python.org/}{\texttt{ Python}} 3.\+x interpreted language ({\ttfamily python}).

A graphical user interface for tracing stellar orbits is provided with the Yorick plug-\/in (see \mbox{\hyperlink{yorick_plugin_page}{Gyoto for Yorick}}).
